## about
timetide is a customizable pomodoro timer designed to enhance focus and productivity.

## features
- start, pause, unpause, reset, and skip functionality
- calendar heat map to show dates active
- self-tracking stats (e.g., timer cycles complete, current streak + highest streak achieved, timers started/paused/unpaused/reset, + more)
- adjustable settings (i.e., enable/disable timer looping, enable/disable sounds, enable/disable notifications, + reset stats)
